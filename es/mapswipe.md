description: |-
  Desde tu navegador web o tu teléfono móvil, puedes ayudar a comunidades de todo el mundo, un deslizamiento a la vez.

  MapSwipe aprovecha la fuerza colectiva de las personas voluntarias para contribuir activamente a proyectos de datos geoespaciales. Desde la identificación de infraestructuras hasta el seguimiento de cambios ambientales y la validación de datos cartográficos, las personas MapSwipers ayudan a mejorar los mapas en todo el mundo.

  MapSwipe es una aplicación móvil y web de código abierto que hace que la cartografía global sea más coordinada y eficiente. Las personas voluntarias deslizan imágenes en distintos tipos de proyectos, cada uno diseñado para responder a necesidades específicas de mapeo.

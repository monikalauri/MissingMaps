# Visión general
Missing Maps se compromete a proteger tu privacidad. Este sitio web es de carácter
informativo y puede utilizarse sin crear una cuenta ni proporcionar información
personal. Solo procesamos datos técnicos limitados necesarios para el funcionamiento
del sitio web y para mostrar contenido de terceros incrustado.

# Información que recopilamos
No recopilamos activamente datos personales como nombres, direcciones de correo
electrónico o cuentas de usuario a través de este sitio web.

Sin embargo, cuando visitas el sitio, cierta información técnica limitada puede ser
procesada automáticamente, incluyendo:
- dirección IP
- información del navegador y del dispositivo

Esta información es procesada por nuestro proveedor de alojamiento y servicios de
terceros, tal como se describe a continuación.

## Alojamiento (GitHub Pages)
Este sitio web está alojado en GitHub Pages. GitHub puede procesar datos técnicos
(como direcciones IP y registros) para proporcionar y asegurar sus servicios de
alojamiento. Este procesamiento se rige por las propias prácticas de privacidad de
GitHub.

## Contenido incrustado de terceros (YouTube)
Algunas páginas de este sitio web incluyen vídeos incrustados de YouTube, un
servicio proporcionado por Google.

Cuando ves o interactúas con un vídeo de YouTube incrustado, Google puede recopilar
información como tu dirección IP y establecer cookies en tu dispositivo. Este
procesamiento está sujeto a la Política de Privacidad de Google.

Utilizamos incrustaciones de YouTube únicamente para mostrar contenido de vídeo
relacionado con nuestras actividades.

## Enlaces a terceros
Nuestro sitio web puede incluir enlaces a sitios web o servicios de terceros (por
ejemplo, YouTube, HOT Tasking Manager, redes sociales). No somos responsables del
contenido ni de las prácticas de privacidad de estos sitios.

# Cookies
Este sitio web no establece cookies por sí mismo. El contenido incrustado de terceros
(como los vídeos de YouTube) puede establecer cookies cuando interactúas con él.
Puedes controlar o eliminar las cookies en cualquier momento a través de la
configuración de tu navegador.

## Cómo utilizamos la información
Cualquier información procesada se utiliza únicamente para:
- garantizar el correcto funcionamiento del sitio web
- mostrar contenido incrustado

No utilizamos datos personales con fines de marketing, elaboración de perfiles ni
comerciales.

# Compartición de datos
No vendemos ni compartimos datos personales.

Los datos pueden ser procesados por terceros únicamente cuando sea necesario para
el funcionamiento del sitio web (por ejemplo, alojamiento y medios incrustados).

# Tus derechos
Si te encuentras en la Unión Europea, tienes derechos en virtud del Reglamento General
de Protección de Datos (RGPD), incluido el derecho a acceder, corregir o solicitar la
eliminación de tus datos personales, cuando corresponda.

# Cambios en esta política
Podemos actualizar esta Política de Privacidad ocasionalmente. Cualquier cambio se
reflejará en esta página con una fecha de revisión actualizada.

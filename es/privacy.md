Hemos hecho todo lo posible por crear una política de privacidad simple y transparente. **Nuestro objetivo es no recopilar datos, o recopilar la menor cantidad posible.** Solo utilizamos cookies esenciales para el funcionamiento de este sitio web.

Este sitio se genera de forma estática y se sirve como HTML. Esto significa que existe una superficie muy reducida para que el sitio sea hackeado o comprometido. No requiere una base de datos ni un servidor backend para procesar o mostrar contenidos. Técnicamente existe un servidor web, pero solo sirve documentos pre-renderizados generados durante la fase de construcción.

---

## Servicios que utilizamos

### Analítica

Este sitio utiliza Plausible Analytics, una alternativa a Google Analytics centrada en la privacidad y conforme al RGPD. Todas las mediciones se realizan de forma completamente anónima. No se utilizan cookies ni se recopilan datos personales. Todos los datos son agregados.  
[Más información](https://plausible.io/privacy-focused-web-analytics)

### Publicidad y marketing

Este sitio no incluye ningún código de seguimiento publicitario o de marketing de terceros.

### Fuentes

Este sitio no utiliza Google Fonts desde el CDN de Google. Utiliza fuentes autoalojadas que no realizan seguimiento.

### YouTube

Cuando se incrustan vídeos de YouTube en este sitio, utilizamos el modo de privacidad mejorada de YouTube.  
[Más información](https://support.google.com/youtube/answer/171780?hl=es#zippy=%2Cturn-on-privacy-enhanced-mode)

### Formulario de contacto

Podemos solicitar información personal a través de nuestro formulario de contacto. Esta información puede incluir, entre otros:

- Dirección de correo electrónico  
- Nombre y apellidos  
- Dirección  
- Número de teléfono  

Los datos enviados pueden compartirse con proveedores externos, incluidos:

- [Formspree](https://formspree.io/legal/privacy-policy/)  
- [Netlify](https://www.netlify.com/privacy/)  

También puedes contactarnos directamente por correo electrónico sin usar el formulario.

### Suscripción al boletín

Podemos solicitar información personal mediante el formulario de suscripción al boletín. Esta información puede incluir:

- Dirección de correo electrónico  
- Nombre y apellidos  

Los datos pueden compartirse con proveedores externos, incluidos:

- [Mailchimp](https://mailchimp.com/legal/)

### Comentarios

Al publicar comentarios en este sitio, tus datos pueden compartirse con un servicio externo, incluido:

- [Disqus](https://disqus.com/privacy-policy/)

## Información que recopilamos

Podemos recopilar los siguientes tipos de datos personales:

- **Información de contacto**: nombre, correo electrónico, dirección postal, número de teléfono  
- **Datos de registro**: información que tu navegador envía al visitar el sitio  
- **Cookies**: archivos de datos almacenados en tu dispositivo que pueden incluir un identificador anónimo  

## Cómo utilizamos tu información

Podemos utilizar la información recopilada para:

- **Proporcionar y mantener el sitio**  
- **Enviar comunicaciones o actualizaciones**  
- **Cumplir obligaciones legales**  

## Compartición de datos

Podemos compartir tus datos personales:

- **Con proveedores de servicios**  
- **Por motivos legales**  

## Tus derechos

- **Rectificar** tus datos  
- **Solicitar la eliminación**  
- **Solicitar la portabilidad de los datos**
